# UNP_Agriculture_crop_yield_in_Indian_State_USING_GEN_AI_AND_PYTHON

This project focuses on predicting and analyzing agricultural crop yields across various Indian states using advanced Time-Series Forecasting models and Python. It addresses the critical challenge of yield variability caused by factors like rainfall dependency, soil fertility, and climate change.
